
# INFLYR Multi-Page Structure

## Structure

/
├── index.html
├── brands.html
├── creators.html
├── faq.html
├── legal.html
│
├── assets/
│   ├── css/style.css
│   └── js/main.js

## Usage

Upload all files to GitHub repo root.

GitHub Pages will automatically work.

## Important

You still need to move actual sections/content from old index.html
into the respective pages.

This structure removes the single-file nightmare and gives proper scaling.

Because apparently humans enjoy turning one HTML file into a geological formation.
